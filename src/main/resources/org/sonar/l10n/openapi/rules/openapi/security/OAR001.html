For security reasons and as a REST best practice, the HTTPS protocol is mandatory.

Noncompliant Code Example (OpenAPI 2)

JSON

{
    "swagger": "2.0", 
    "info": {
        "title": "Swagger Petstore", 
        "version": "1.0"
    }, 
    "schemes": [
        "http"
    ], 
    "paths": {}
}

YAML

swagger: "2.0"
info:
  title: Swagger Petstore
  version: "1.0"
schemes:
  - http
paths: {}

Compliant Solution (OpenAPI 2)

JSON

{
    "swagger": "2.0", 
    "info": {
        "title": "Swagger Petstore", 
        "version": "1.0"
    }, 
    "schemes": [
        "https"
    ], 
    "paths": {}
}

YAML

swagger: "2.0"
info:
  title: Swagger Petstore
  version: "1.0"
schemes:
  - https
paths: {}
