String parameters should have minimum length, maximum length, regular expression, or enum restriction.

Noncompliant Code Example (OpenAPI 2)

  swagger: '2.0'
  info:
    version: 1.0.0
    title: My API
  paths:
    /items:
      get:
        summary: Get a list of items
        parameters:
          - name: param1
            in: query
            type: string
        responses:
          200:
            description: A list of items
            schema:
              type: array
              items:
                type: string

Compliant Solution (OpenAPI 2)

  swagger: '2.0'
  info:
    version: 1.0.0
    title: My API
  paths:
    /items:
      get:
        summary: Get a list of items
        parameters:
          - name: param1
            in: query
            type: string
            minLength: 1
            maxLength: 10
            enum: ['option1', 'option2']
        responses:
          200:
            description: A list of items
            schema:
              type: array
              items:
                type: string
