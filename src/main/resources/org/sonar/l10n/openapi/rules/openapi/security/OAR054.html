Hostname must be a subdomain of the organization's domain name.

Noncompliant Code Example (OpenAPI 2)

JSON

{
    "swagger": "2.0", 
    "info": {
        "version": "1.0.0", 
        "title": "Swagger Petstore"
    }, 
    "host": "other.domain.com", 
    "paths": {}
}

YAML

swagger: "2.0"
info:
    version: 1.0.0
    title: Swagger Petstore
host: other.domain.com
paths: {}

Compliant Solution (OpenAPI 2)

JSON

{
    "swagger": "2.0", 
    "info": {
        "version": "1.0.0", 
        "title": "Swagger Petstore"
    }, 
    "host": "api.cloudappi.net", 
    "paths": {}
}

YAML

swagger: "2.0"
info:
    version: 1.0.0
    title: Swagger Petstore
host: api.cloudappi.net
paths: {}

Noncompliant Code Example (OpenAPI 3)

JSON

{
    "openapi": "3.0.0", 
    "info": {
        "version": "1.0.0", 
        "title": "Swagger Petstore"
    }, 
    "servers": [
        {
            "url": "https://aws-dev-cloudappi.net/api-users/v1/"
        }, 
        {
            "url": "https://aws-test.cloudappi.net/api-users/v1/"
        }, 
        {
            "url": "https://aws-cloudappi.net/api-users/v1/"
        }
    ], 
    "paths": {}
}

YAML

openapi: "3.0.0"
info:
    version: 1.0.0
    title: Swagger Petstore
servers:
    - url: https://aws-dev-cloudappi.net/api-users/v1/
    - url: https://aws-test.cloudappi.net/api-users/v1/
    - url: https://aws-cloudappi.net/api-users/v1/
paths: {}
