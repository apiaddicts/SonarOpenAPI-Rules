En operaciones con respuesta paginada, el parámetro $limit, que sirve para indicar el tamaño de la página, se debe definir como parámetro no obligatorio.

Ejemplo de código no compatible (OpenAPI 2)

swagger: "2.0"
info:
  title: Swagger Petstore
  version: 1.0.0
paths:
  /pets:
    get:
      responses:
        200:
          description: Ok

Solución compatible (OpenAPI 2)

swagger: "2.0"
info:
  title: Swagger Petstore
  version: 1.0.0
paths:
  /pets:
    get:
      parameters:
        - in: query
          name: $limit
          required: false
          type: integer
          default: 25
      responses:
        200:
          description: Ok
